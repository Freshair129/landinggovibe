Common Asked

FAQs

หาสิ่งที่ต้องการไม่เจอใช่ไหม? ทีมเราพร้อมตอบคำถามเฉพาะโปรเจกต์ของคุณ

ตอบกลับภายใน [เวลา]

GoVibe แทนที่ Claude Code / Gemini CLI ของผมไหม?

ไม่ — GoVibe เป็นชั้นประสานงาน ไม่จัดการ billing หรือแทนที่เครื่องมือ coding ภายนอก คุณใช้ของเดิมต่อได้

ข้อมูลบนบอร์ดมาจากไหน?

มาจากไฟล์เอกสาร Markdown/HTML ที่อนุมัติแล้วใน docs/roadmap/ — ไม่มี mock data ตามกฎ live-data-only

ต้องเขียน Genesis atom เองไหม?

ไม่ต้อง — atom ถูกสกัดจากเอกสาร SWE ที่อ่านได้สำหรับมนุษย์โดยอัตโนมัติหลังเขียนเสร็จ

external executor อนุมัติงานเองได้ไหม?

ไม่ได้ — output ของ executor ภายนอกเป็นงานร่างเสมอ ต้องผ่าน lead review, QA และ audit ก่อนปิดงาน

รันแบบ local ได้ไหม?

ได้ — มี HTTP/WS sidecar ที่ 127.0.0.1:4310 รองรับ Ollama local runtime ไม่ผูกกับ cloud
